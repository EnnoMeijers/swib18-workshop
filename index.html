Application Programming Interfaces

Why API?

APIs make software development manageable

modular software

stable applications

flexible data sources

e.g. Lobid API data sources and formats

APIs decouple applications from specific data sources, formats, and systems. They enable modular, sustainable applications.

Usable data

= data with APIs

JSON

cURL

Exercise 1:
cURL


# Get your own profile
$ curl https://api.github.com/users/<user>

# Get data for a repo
$ curl https://api.github.com/repos/hbz/swib18-workshop

Last cURL output was long, we often want specific values

jq

A lightweight and flexible command-line JSON processor

https://stedolan.github.io/jq/

https://jqplay.org/

Exercise 2:
Using the GitHub API

GET 

$ curl https://api.github.com/repos/hbz/swib18-workshop
Name 

$ curl ... | jq '.name'
License 

$ curl ... | jq '???' # get the repo's license name
Issues 

$ curl ... | jq '???' # get the number of open issues

('$' means you should type this in your terminal application)
