Context

Maps JSON keys to URIs
"name": "http://schema.org/name"

Here: generated from ontology

Context can do more than simple name-URI mapping


"contribution": {
  "@id": "http://id.loc.gov/ontologies/bibframe/contribution",
  "@container": "@set"
}

input['@graph'].map(oldDoc => {
  const newDoc = {
    '@context': 'http://lobid.org/bibframe-context.jsonld'
  };
  for(x in oldDoc) { newDoc[x] = oldDoc[x]; }
  return newDoc;
